# QVSS genomic interval profile note

This note documents the profile identifier used by examples that apply `overlaps` to genomic intervals.

```yaml
id: https://qvss.org/profiles/genomic-interval/1.0
name: qvss-genomic-interval
version: 1.0.0
```

A QV set using interval overlap should declare the reference assembly, coordinate convention, minimum overlap requirement, and resource provenance. This note is illustrative and does not replace the normative QVSS text.
