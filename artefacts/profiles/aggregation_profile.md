# QVSS aggregation profile note

This note documents the profile identifier used by examples that require grouped or counted records.

```yaml
id: https://qvss.org/profiles/aggregation/1.0
name: qvss-aggregation
version: 1.0.0
```

Aggregation semantics are outside the core QVSS rule language. A production profile should define grouping, counting, record scope, missing-value behaviour, and expected outputs.
